# Active Tasks

Shared WIP board for all agents operating on this project.

Register a task before starting a non-trivial change. Mark `[DONE]` or remove when
finished so other agents know the lock is released.

Format:

```
- [ ] <alias> — <short description> — started <YYYY-MM-DD HH:MM> — files: <paths>
```
