# UI Component Rules

- Dark theme (zinc-900 base), MD/markdown aesthetic, indie/amateur style
- Pixel art characters: `imageRendering: 'pixelated'`
- Floating characters: position near mouse (120~180px offset), never block click areas
- All components use `'use client'` directive when using hooks/state
- Tailwind 4 classes only, no inline styles
- English UI text default (i18n-ready)
