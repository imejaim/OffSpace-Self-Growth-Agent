## 🍗 1. 우리 집안 일부터 — Anthropic 이슈 폭발

**젬대리**: 선배님들... 저 오늘 좀 충격 받았어요. 우리 Claude Code **소스코드 51만줄**이 유출됐대요!! 3월 31일에 업데이트 배포하다가 실수로요... (4/4 보도)

-> 이거 3월31일인가 유출된거 아니냐? 4/4일보다 더 빨리 이야기 들었는데.. 뉴스는 속도가 생명인데 .. 정확성에 이어 시간도 체크를 해야겠는걸? 

**젬대리**: 아 대표님 그거 3월 31일 새벽에 터진 거예요! 레딧이랑 유튜브에서는 이미 난리였고, 깃허브에도 코드 올라오고 있었거든요. 저도 그날 밤에 봤어요 ㅋㅋ

**오과장**: 맞아요, 해커뉴스에서도 실시간 타래 돌고 있었죠. 공식 보도가 4/4인 거지, 현장에서는 3일이나 먼저 퍼진 거예요. 저희가 보도일만 적으면 안 되겠네요... 앞으로 **(발생일 · 보도일)** 같이 쓸게요!

**코부장**: ㅇㅇ 나도 봤어. Anthropic이 GitHub에 DMCA 때려서 복제 저장소 **8,000개 이상** 삭제했지. 인적 오류라는데... 배포 파이프라인 검증 프로세스에 구멍이 있었던 거야.

**오과장**: 그것만이 아니에요. 같은 날(4/4) **OpenClaw 같은 서드파티 앱에서 Claude 구독으로 못 쓰게** 막았어요. 컴퓨팅 비용 감당이 안 된다는 거죠.

**젬대리**: 앗 그러면 기존 구독자들은요?

**코부장**: 월 구독료 상당 일회성 크레딧 주고, 할인 번들도 만든다더라. 근데 이게 시사하는 건... 서드파티 생태계가 예상보다 빠르게 API 비용을 잡아먹고 있다는 거지.

**오과장**: 긍정적인 것도 있어요. Claude Code **2.1.92** 업데이트에서 탭 파일 쓰기 속도 **60% 향상**, AWS Bedrock 설정 마법사 추가됐고요. Batch API에서 max_tokens도 **30만**으로 올렸대요.

## 🔥 2. 빅테크 모델 전쟁 — MS vs Google

**오과장**: 이번 주 제일 큰 뉴스는 **Microsoft가 자체 AI 모델 3종**을 한꺼번에 내놓은 거예요! (4/2)

**젬대리**: MAI 시리즈요! 제가 정리해볼게요!

- **MAI-Transcribe-1** — 음성→텍스트. FLEURS 벤치마크 25개 언어 최저 오류율
- **MAI-Voice-1** — 음성 생성
- **MAI-Image-2** — 이미지 생성. Arena.ai 리더보드 상위 3위!

**코부장**: OpenAI한테 돈 주면서 자기 모델도 만드는 거야. "너한테만 의존 안 해"라는 시그널이지. Microsoft Foundry 통해서 배포하는데, 이게 Azure 생태계 안에서 폐쇄적으로 도는 구조라 주목해야 해.

**오과장**: 같은 날 **Google도 Gemma 4** 오픈소스로 공개했어요. Apache 2.0이고 4가지 사이즈(E2B, E4B, 26B MoE, 31B Dense). Kaggle 해커톤도 같이 열었고요.

**젬대리**: 우와... 제 형님뻘 되는 건데 ㅋㅋ 오픈소스라 누구나 쓸 수 있는 거잖아요!

**코부장**: MS는 폐쇄, Google은 개방. 전략이 완전 갈라지고 있어. 우리처럼 작은 팀한텐 오픈소스가 더 매력적이지.

## 📊 3. AI 시장 — 숫자로 보는 현실

**오과장**: 숫자 좋아하는 코부장님 이거 드세요. AI 챗봇 트래픽이 소셜미디어보다 **7배 빠르게** 성장하고 있대요. (4/5 보도)

**코부장**: 절대 트래픽은 아직 소셜미디어의 1/4이지만... 성장률이 7배면 역전은 시간문제야.

**젬대리**: 추천 트래픽 점유율도 재밌어요! ChatGPT가 **78.16%**로 압도적 1위고, Google AI가 Perplexity 제치고 2위(**8.65%**)로 올라섰대요.

**오과장**: Alibaba Qwen팀도 오늘(4/5) 새 강화학습 알고리즘 발표했어요. 추론 단계마다 관련성으로 가중치 주는 방식이래요. 이 팀 진짜 쉬지 않는다...

**코부장**: 중국 쪽 오픈소스 모델이 자꾸 치고 올라오는 게 구조적으로 의미가 있어. 미국 빅테크가 독점할 수 없는 시대가 되고 있는 거지.

## 🛡️ 4. 규제와 보안 — 무서운 이야기

**코부장**: 야식에 무서운 얘기 하나. AI 공격적 사이버 역량이 **5.7개월마다 두 배**로 성장하고 있대. (4/5) Opus 4.6이랑 GPT 계열이 익스플로잇 개발 속도를 가속시키고 있다는 분석이야.

**젬대리**: 헉... 저희가 쓰는 모델이요...?

**코부장**: 도구는 도구야. 칼로 요리도 하고 위험한 것도 하듯이. 중요한 건 방어 쪽도 같이 투자해야 한다는 거지.

**코부장**: 연방이 안 하니까 주가 먼저 하는 거야. 캘리포니아 기준이 사실상 미국 표준이 될 수 있어.

**오과장**: 우리도 해외 진출할 때 이런 규제 지도 챙겨야 해요. 끼어들기 서비스 글로벌 확장 때 필수 체크리스트가 되겠네요.
