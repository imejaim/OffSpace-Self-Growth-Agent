# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**OffSpace Self-Growth Agent** — Offspace 회사의 자율 진화 에이전트 시스템. 테스형(12_테스형)이 독립된 에이전트를 만드는 프로젝트라면, 이 프로젝트는 **에이전트 자체가 스스로 진화**하는 시스템이다.

2026년 4월 2일, Offspace에 코부장·오과장·젬대리 직원 구성을 기념하여 시작됨.

## Core Concept

1. **AI 동향 파악** — 매일 최신 AI 동향을 수집·분석하여 유용한 기술/도구를 식별
2. **자율 진화 방향 설정** — 수집된 정보를 바탕으로 스스로 진화 방향을 결정
3. **승인 프로세스** — 진화 방향이 설정되면 초천재 대표님에게 보고·승인 요청
4. **자가 평가** — 진화 결과를 스스로 평가하고 유지/롤백 판단
5. **콘텐츠 발행** — 매일 파악한 정보를 코부장·오과장·젬대리 3인의 티타임 대화 형식으로 정리하여 티스토리 블로그에 게시
6. **직원 캐릭터** — CLI 환경에서 각 직원(코부장, 오과장, 젬대리)을 표현할 캐릭터 시스템

## Language

- 문서, 커밋 메시지, UI 텍스트는 한국어 사용
- 코드 내 변수/함수명은 영어 사용

## 티타임 발행 필수 규칙 (MANDATORY)

티타임/야식타임 콘텐츠 작성 시 아래 규칙을 **반드시** 따른다. 상세 가이드는 `output/teatime/TEATIME_RULES.md` 참조.

### 날짜/요일 검증
- 작성 전 반드시 `python scripts/teatime-skeleton.py YYYY-MM-DD`로 요일을 확인한다
- 완성 후 반드시 `python scripts/teatime-skeleton.py --validate <파일경로>`로 검증한다
- **검증 통과 전 발행 금지**

### 고정 카테고리 (순서/이름 변경 금지)
1. **AI 핫뉴스** — 빅테크 동향, 투자/인수/IPO, 경영진 변동
2. **AI 에이전트** — 에이전트 프레임워크, 프로토콜 표준, 배포 사례 (IPO/인사 금지)
3. **AI 논문과 모델** — 모델 릴리즈, 벤치마크, 오픈소스, 추론 인프라
4. **AI 로봇 / 피지컬 AI** — 휴머노이드, 자율주행, 온디바이스 HW
5. **보너스 / 그 외** — 규제, 사회 이슈, 재미

형식: `## N. [카테고리] — "서브타이틀"`

### 출처/이미지 최소 기준
- 토픽당 링크 **3개+**, 전체 **12개+**, SNS/커뮤니티(Reddit, YouTube, X.com, HN) **2개+**
- 이미지 **최소 2장** — 해당 뉴스 기사의 실제 대표 이미지 사용 (기존 이미지 재활용 금지, AI 생성 금지)
- (발생일 · 보도일) 병기 필수

### 캐릭터 채널 분담
- **젬대리**: Reddit, YouTube, X.com, GitHub → 커뮤니티 캐치
- **오과장**: HackerNews, Crunchbase, 시장 보고서 → 팩트/숫자 보강
- **코부장**: 공식 블로그, 논문, 기술 문서 → 기술 분석, 마무리
