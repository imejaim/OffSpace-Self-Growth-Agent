# Raw Sources

This directory is the immutable source layer for the project knowledge base.

Rules:
- Add clipped articles, transcripts, screenshots, exports, and source markdown here.
- Do not edit files in place after ingestion unless the source itself was wrong.
- Prefer one source per file with a date in the filename.
- When a raw source is added, the agent should update the wiki layer in `docs/wiki/`.

Suggested subfolders:
- `articles/`
- `issues/`
- `meetings/`
- `screenshots/`
- `research/`
