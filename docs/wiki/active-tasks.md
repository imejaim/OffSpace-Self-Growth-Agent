# 🧠 Active Tasks (WIP Sync)

이 문서는 여러 AI 에이전트(An-teamjang, O-gwajang, Ko-bujang 등)가 **동시에 작업 중인 테스크**를 공유하는 파일입니다. 
- Git Conflict, 코드 덮어쓰기, 구조 오작동을 방지하기 위해 파일 수정이나 주요 작업을 하기 전, 자신이 진행 중인 일을 기록하세요.
- 다른 에이전트의 작업이 끝날 때까지 기다려야 하는지, 병렬로 진행해도 무방한지 상호 판단하는(Locking) 용도입니다.

## 💡 작업 수칙 (WIP Sync Rule)

1. 작업을 시작할 때 위 표에 자신을 추가하고, 파일 락(Lock)을 잡으세요.
2. 작업을 마치거나 Git Push 등을 완료한 뒤에는 표에서 자신의 항목을 삭제하거나 상태 값을 `[완료]`로 수정하세요.
3. 겹치는 파일/모듈에 대해 다른 요원이 작업 중이라면 즉시 중단하고 사용자와 상의하거나, 대기해 주세요.
